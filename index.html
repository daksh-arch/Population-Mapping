Human Terrain

An interactive 3D map of global population density. Each hexagon represents ~36 km² at mid zoom. Height and colour show how many people live there.

Inspired by Matt Daniels' Human Terrain on The Pudding.

Data

Population counts from Kontur Population 2023 — a global H3 hexagon grid derived from multiple census and satellite sources.

How it's built

MapLibre GL JS + PMTiles. Vector tiles processed with tippecanoe at two H3 resolutions (r6 at z3–5, r7 at z6–9), served as a single 688 MB PMTiles file from Cloudflare R2.
